# ClassWork

## 1.OdCNN  （一维卷积与池化）

依照图示与相关资料，形成程序基本框架：

1.给出最大数据数量，由用户给出数据和卷积核，使用一维数组存储（实现了数据个数不定的数据读入）。

2.按照算法进行卷积：依次读取数据数组中与卷积核数组相同长度的子数组与卷积核数组进行卷积运算，并另外定义一个一维数组和一个变量存储中间数据并统计数据个数。

3.输出中间数据验证卷积运算的准确性。

4.由用户给出池化过滤器大小。

5.对卷积结果进行池化：依次读取中间数据数组中与池化过滤器相同大小的子数组，将子数组数据传入临时数组并进行降序排序，将临时数组第一个元素（即最大值）存入结果数组，并定义了一个变量统计池化结果个数。

6.输出池化结果，程序结束

（7.将卷积运算和池化运算包装为两个函数。）

程序实际编写时使用了qsort()及其所需的比较函数。

## 2.TdCNN  （二维卷积与池化）

依照图示与相关资料，形成程序基本框架：

1.给出最大矩阵大小，由用户给出数据和卷积核及其大小，使用二维数组存储。

2.按照算法进行卷积：依次读取数据数组中与卷积核数组相同大小的子二维数组与卷积核数组进行卷积运算，并另外定义一个二维数组和一个一维数组存储中间数据矩阵并统计矩阵大小。

3.输出中间数据验证卷积运算的准确性。

4.由用户给出池化过滤器大小。

5.对卷积结果进行池化：依次读取中间数据数组中与池化过滤器相同大小的子二维数组，将子二维数组数据以一维形式传入临时数组并进行降序排序，将临时数组第一个元素（即最大值）存入结果二维数组，并定义了一个一维数组统计池化结果个数。

6.输出池化结果，程序结束

（7.将求二维数组最大值、卷积运算和池化运算包装为三个函数。）

程序实际编写时使用了qsort()及其所需的比较函数，使用了指针将二维数组传入函数并读取和修改数组元素。

## 3.Sqrt  （平方根计算）

基本思路：

将平方根的计算转化为求f(x)=x^2-q的零点（q为被开方数）

求零点使用的算法为[牛顿迭代法](https://baike.baidu.com/item/%E7%89%9B%E9%A1%BF%E8%BF%AD%E4%BB%A3%E6%B3%95?fromModule=lemma_search-history)。

使用clock()等time.h下的数据类型和函数进行计算时间的统计，以及与sqrt()函数运算速度的比较。

## 4.Integral  （积分计算）

基本思路：

从几何角度分析，将对应图形分割为左边的曲边梯形和右边的三角形，并分别计算其面积。

问题的难点在于分割点的横坐标不可直接表示，于是考虑使用类似求零点的方法来计算该值。

## 5.Min  （函数最小值）

从函数式分析，可能的最小值为-7，即当sin(x^2)=-1，sin x=1时取得该最小值，问题则转化为找出符合条件的x。

显然2式的解集是容易求得并且有规律的（以2π递增），则问题继续简化为在该解集中寻找1式的解，程序思路已经呼之欲出了。

（在程序编写及调试中注意到，将x对应的变量存储在循环内和循环外会得到不同的结果，猜测为浮点数进行加法运算时产生了数据丢失。）
